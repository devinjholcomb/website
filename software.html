How to Install SPEC

This section explains how to install and run the SPEC application on a new machine.

Prerequisites

  • A computer running Linux or macOS (or WSL on Windows).
  • Git installed so you can clone the repository.
  • A terminal that can run Bash shell scripts.

1. Clone the SPEC repository

  1. Open a terminal.
  2. Choose a folder where you want to keep the project.
  3. Clone the repo

2. Run the setup script

All required software and libraries are installed by a single setup script that lives in the setup/ directory of the project.

  1. From the project root, change into the setup folder: 
    cd setup
  2. Run the script: 
    ./set_up_system.sh

The set_up_system.sh script will automatically download and install all dependencies needed to run SPEC. This includes the programming language runtime, required libraries, and any command-line tools that the application expects. The script is designed so that a new user does not have to install each dependency manually.

3. What the setup script installs

Internally, set_up_system.sh installs the following categories of software (the exact list is documented in the script itself):

  • Core runtime.
  • Libraries for camera access, video processing, and numerical computing.
  • Any additional utilities used by the SPEC pipeline.

All of these come from standard package repositories

4. Verifying the installation

  1. Return to the project root: 
    cd ..
  2. Change the current directory to app and then run the main application command: 
    cd app & python3 app.py

If the installation succeeded, the SPEC application should start without errors and display the main interface or command-line help message.
