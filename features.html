Trapezoid Calibration: When you first sign in to the flask application click on calibrate trapeoid points, from there you can adjust the x and y axis as you need. There are also additional instructions.

Piv Functions

Run Test: Click Run test and it will run one loop of the the piv capture system

Run Piv: Click Run Piv and it will run one loop of the piv capture system

Start Auto Piv: Click start auto piv and this will start the automatic capturing of the piv through a continuious loop

Stop Auto Piv: Click stop auto piv and this will stop the automatic capturing of the piv
